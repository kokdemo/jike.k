**即刻的网页版重新写了一版新的，原来的特性已经不生效了，新的到底要怎么写好没有想好**


chrome官方商店安装：[点我](https://chrome.google.com/webstore/detail/jikek/ceohoefapneebmoilaicplfenfkmfoni)
